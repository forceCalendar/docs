## SearchOptions

| Option | Type | Default |
|--------|------|---------|
| `fields` | `string[]` | `['title', 'description', 'location']` |
| `fuzzy` | `boolean` | `false` |
| `caseSensitive` | `boolean` | `false` |
| `limit` | `number` | `50` |
| `sortBy` | `string` | `'relevance'` |

| Option | Type |
|--------|------|

| `locations` | `string[]` |
| `attendees` | `string[]` |
| `status` | `string[]` |
| `allDay` | `boolean` |
| `recurring` | `boolean` |
| `hasReminders` | `boolean` |
| `custom` | `(event: Event) => boolean` |
